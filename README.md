# kelly-valencia-portfolio
Data science and machine learning projects, including forecasting models, data cleaning, and real-world analysis using R and Python.

## 1. Unemployment Forecasting: Time Series & ML
Built and compared multiple forecasting models, including ARIMA, ARIMAX, Ridge, Lasso, and regression trees, using U.S. macroeconomic data.  
Evaluated model performance using out-of-sample RMSE and MAE, and tested robustness during the COVID-19 period.

---

## 2. Data Cleaning & Exploratory Analysis: Quality of Life
Cleaned and transformed a real-world dataset with missing values, inconsistent formatting, and placeholder data.  
Converted variables into usable formats and performed exploratory analysis to identify patterns across countries.

---
## 3. Interest Rates & Stock Return Predictability (PCA)
Analyzed whether Treasury yield curve movements can predict S&P 500 returns using principal component analysis (PCA) and regression models.  
Found that predictive power is extremely limited, supporting the efficient market hypothesis.
